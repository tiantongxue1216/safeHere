# 匿名倾诉社区小程序

一个温暖、安全的匿名倾诉社区，为需要倾听、鼓励和安慰的人提供一个可以安心表达的空间。

## 功能特点

### 核心功能
- ✅ **匿名发帖** - 无需登录，完全匿名
- ✅ **文字/语音/表情** - 多种表达方式
- ✅ **一键鼓励** - 温暖的蓝色爱心按钮
- ✅ **匿名评论** - 传递温暖和支持
- ✅ **我的帖子** - 查看自己的心路历程

### AI 内容审核（千问大模型）
- ✅ **黄色暴力检测** - 拦截违规内容
- ✅ **伤害性言论检测** - 保护发帖人情感
- ✅ **自动过滤** - 营造温暖社区氛围

### 设计理念
- 🎨 **简洁页面** - 无多余功能，专注倾听
- 🎨 **蓝色冷色调** - 让发帖人冷静舒适
- 🎨 **温暖交互** - 每一个细节都传递关怀

## 技术架构

### 前端
- 微信小程序原生开发

## 快速开始

### 1. 环境准备

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号（用于获取 AppID）
3. 开通微信云开发服务

### 2. 配置项目

#### 2.1 填写 AppID
在 `project.config.json` 中填入你的小程序 AppID：
```json
{
  "appid": "你的小程序 AppID"
}
```

#### 2.2 配置云环境
在 `app.js` 中修改云环境 ID：
```javascript
wx.cloud.init({
  env: 'your-cloud-env-id', // 替换为你的云环境 ID
  traceUser: false
})
```

#### 2.3 配置千问 API Key
在以下两个文件中填入你的阿里云千问 API Key：

- `cloudfunctions/createPost/index.js`
- `cloudfunctions/createComment/index.js`

```javascript
const QWEN_API_KEY = 'YOUR_QWEN_API_KEY' // 替换为你的 API Key
```

**获取千问 API Key：**
1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 开通 DashScope 服务
4. 创建 API Key

### 3. 部署云函数

1. 在微信开发者工具中，右键点击 `cloudfunctions/createPost` 目录
2. 选择"上传并部署：云端安装依赖"
3. 同样步骤部署 `cloudfunctions/createComment`

### 4. 创建数据库

在微信开发者工具的"云开发"控制台中创建以下集合：

#### posts 集合
- `_id`: 自动生成的记录 ID
- `userId`: 用户 ID（OpenID）
- `content`: 帖子文字内容
- `voiceUrl`: 语音文件 ID
- `encourageCount`: 鼓励数量（数字，默认 0）
- `commentCount`: 评论数量（数字，默认 0）
- `createTime`: 创建时间
- `isDeleted`: 是否已删除（布尔值）

#### comments 集合
- `_id`: 自动生成的记录 ID
- `postId`: 关联的帖子 ID
- `userId`: 用户 ID（OpenID）
- `content`: 评论内容
- `voiceUrl`: 评论语音文件 ID
- `createTime`: 创建时间
- `isDeleted`: 是否已删除（布尔值）

#### encourages 集合
- `_id`: 自动生成的记录 ID
- `postId`: 关联的帖子 ID
- `userId`: 用户 ID（OpenID）
- `createTime`: 创建时间

**创建索引：**
- posts 集合：为 `userId`、`createTime` 字段创建索引
- comments 集合：为 `postId`、`createTime` 字段创建索引
- encourages 集合：为 `postId`、`userId` 创建联合索引

### 5. 运行项目

1. 在微信开发者工具中打开项目
2. 编译运行即可看到效果

## 使用说明

### 发贴流程
1. 点击首页"💌 写下你的心声"按钮
2. 输入文字内容（可选）
3. 长按"按住说话"录制语音（可选）
4. 选择表情符号（可选）
5. 点击"发布心声"
6. 系统自动审核内容（约 1-2 秒）
7. 发布成功，返回首页

### 鼓励流程
1. 进入任意帖子详情页
2. 点击"一键鼓励"蓝色爱心按钮
3. 按钮变粉色并播放心跳动画
4. 鼓励数 +1
5. 再次点击可取消鼓励

### 评论流程
1. 在帖子详情页底部输入框输入评论
2. 或点击麦克风录制语音评论
3. 点击"发送"
4. 系统自动审核评论内容
5. 审核通过后显示评论

## 审核规则

### 帖子审核规则
以下内容将被拦截：
- ❌ 色情、淫秽内容
- ❌ 暴力、血腥内容
- ❌ 违法违纪内容
- ❌ 其他违规内容

### 评论审核规则
以下内容将被拦截：
- ❌ 攻击、嘲讽、贬低
- ❌ 冷漠、讽刺、风凉话
- ❌ 说教、指责
- ❌ 任何可能伤害发帖人的言论
- ❌ 违规内容

**只允许温暖、支持、理解的评论通过** ✅

## 注意事项

1. **语音功能**：需要在真机上测试，模拟器不支持录音
2. **API 费用**：千问 API 会产生费用，请注意用量
3. **云开发配额**：免费版云开发有配额限制，超出需升级
4. **内容安全**：建议定期查看帖子和评论内容

## 常见问题

### Q: 如何获取千问 API Key？
A: 访问阿里云百炼平台，注册账号并开通 DashScope 服务即可创建 API Key。

### Q: 云函数部署失败？
A: 检查是否安装了 Node.js，并在云函数目录下右键选择"云端安装依赖"。

### Q: 数据库权限错误？
A: 在云开发控制台设置 posts、comments、encourages 集合的权限为"所有用户可读写"。

### Q: 语音无法播放？
A: 真机测试时需要在微信公众平台配置 downloadFile 合法域名。
