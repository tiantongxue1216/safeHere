## ✨ 核心特性

### 1. 微信风格布局

- **圆角设计**：所有按钮和输入框都采用圆角
- **浅灰背景**：输入栏背景为浅灰色（#f7f7f7）
- **白色输入框**：输入框为白色圆角矩形
- **渐变发送按钮**：激活时使用蓝色渐变

### 2. 交互优化

#### 语音按钮 🔊
- **默认状态**：白色圆形按钮，蓝色语音图标
- **录音状态**：红色背景，红色圆点图标，脉冲动画
- **按压效果**：缩小 95%，背景变深

#### 输入框 📝
- **自动高度**：根据内容自动调整高度
- **最大高度**：300rpx，超出可滚动
- **圆角边框**：40rpx 圆角，柔和美观
- **占位符颜色**：浅灰色（#b0b0b0）

#### 发送按钮 📤
- **禁用状态**：灰色背景，半透明
- **激活状态**：蓝色渐变背景（#667eea → #764ba2）
- **按压效果**：缩小 95%，透明度 90%
- **平滑过渡**：0.3s 过渡动画

#### 表情按钮 😊
- **预留功能**：目前仅显示，后续可扩展
- **样式统一**：与语音按钮保持一致

### 3. 录音提示

录音时底部显示黑色半透明提示条：
```
🎤 松开发送语音
```

- **波浪动画**：麦克风图标上下波动
- **黑色背景**：rgba(0, 0, 0, 0.7)
- **圆角设计**：40rpx 圆角
- **白色文字**：清晰可见

## 🎬 动画效果

## 💡 交互逻辑

### 状态管理

#### 1. 发送按钮状态

```javascript
// 有内容时激活
disabled="{{!(commentContent || tempCommentVoicePath)}}"

// 添加 active 类
class="send-btn {{commentContent || tempCommentVoicePath ? 'active' : ''}}"
```

**逻辑：**
- ✅ 有文字或语音 → 激活（蓝色渐变）
- ❌ 无内容 → 禁用（灰色）

---

#### 2. 录音状态

```javascript
// 录音中
class="voice-btn {{isRecordingComment ? 'recording' : ''}}"

// 显示不同图标
wx:if="{{!isRecordingComment}}" → 🔊
wx:else → 🔴
```

**逻辑：**
-  未按压 → 蓝色语音图标
- 🔴 按压中 → 红色圆点图标 + 脉冲动画

---

#### 3. 录音提示显示

```xml
wx:if="{{isRecordingComment}}"
```

**逻辑：**
- 录音时显示提示条
- 松开后自动隐藏
