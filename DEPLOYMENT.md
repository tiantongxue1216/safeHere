# 快速部署指南 🚀

## 第一步：准备工作（10 分钟）

### 1.1 注册微信小程序
1. 访问 https://mp.weixin.qq.com/
2. 注册小程序账号
3. 获取 AppID（在"开发" -> "开发管理" -> "开发设置"中查看）

### 1.2 开通云开发
1. 打开微信开发者工具
2. 创建新项目（选择"小程序" -> "不使用云服务"）
3. 点击工具栏"云开发"按钮
4. 开通云开发服务（免费版即可）
5. 记录环境 ID（格式类似：cloud1-xxx）

### 1.3 申请阿里云千问 API Key
1. 访问 https://bailian.console.aliyun.com/
2. 注册/登录阿里云账号
3. 开通 DashScope 服务
4. 创建 API Key（会发送短信通知）
5. 保存 API Key

## 第二步：导入项目（5 分钟）

### 2.1 导入代码
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择本项目目录
4. 填入你的小程序 AppID
5. 点击"导入"

### 2.2 修改配置
打开 `app.js`，修改云环境 ID：
```javascript
wx.cloud.init({
  env: 'your-cloud-env-id', // 改为你的云环境 ID
  traceUser: false
})
```

## 第三步：配置云函数（10 分钟）

### 3.1 安装依赖
在微信开发者工具中：
1. 右键点击 `cloudfunctions/createPost` 文件夹
2. 选择"在终端中打开"
3. 执行命令：`npm install`
4. 同样步骤为 `cloudfunctions/createComment` 安装依赖

### 3.2 配置 API Key
打开以下两个文件，替换 API Key：
- `cloudfunctions/createPost/index.js` 第 12 行
- `cloudfunctions/createComment/index.js` 第 13 行

```javascript
const QWEN_API_KEY = 'YOUR_QWEN_API_KEY' // 替换为你的真实 API Key
```

### 3.3 上传云函数
1. 右键点击 `cloudfunctions/createPost` 文件夹
2. 选择"上传并部署：云端安装依赖"
3. 等待上传完成（约 1-2 分钟）
4. 同样步骤上传 `cloudfunctions/createComment`

## 第四步：创建数据库（5 分钟）

### 4.1 创建集合
1. 点击微信开发者工具的"云开发"按钮
2. 进入"数据库"标签
3. 点击"添加集合"
4. 创建以下三个集合：
   - `posts`
   - `comments`
   - `encourages`

### 4.2 设置权限
在每个集合的"权限设置"中：
- 选择"自定义安全规则"
- 粘贴 `database-config.json` 中的对应规则

**posts 集合规则：**
```json
{
  "read": true,
  "write": "auth.openid == doc.userId"
}
```

**comments 集合规则：**
```json
{
  "read": true,
  "write": "auth.openid == doc.userId"
}
```

**encourages 集合规则：**
```json
{
  "read": true,
  "write": "auth.openid == doc.userId"
}
```

### 4.3 创建索引
在每个集合中点击"索引"标签，添加以下索引：

**posts 集合：**
- userId（升序）
- createTime（降序）

**comments 集合：**
- postId（升序）
- createTime（升序）

**encourages 集合：**
- postId（升序）
- userId（升序）

## 第五步：测试运行（10 分钟）

### 5.1 编译预览
1. 点击微信开发者工具的"编译"按钮
2. 查看模拟器效果
3. 检查控制台是否有错误

### 5.2 功能测试
按顺序测试以下功能：

#### 测试发帖
1. 点击"💌 写下你的心声"
2. 输入测试文字："今天心情不太好，想找人说说"
3. 点击"发布心声"
4. 应该显示"发布成功"并返回首页
5. 能看到刚发布的帖子

#### 测试鼓励
1. 点击进入刚才的帖子
2. 点击"一键鼓励"蓝色按钮
3. 按钮应变为粉色并有心跳动画
4. 鼓励数应该 +1

#### 测试评论
1. 在评论区输入："抱抱你，一切都会好起来的 💙"
2. 点击"发送"
3. 应该显示"评论成功"
4. 评论出现在列表中

#### 测试审核
尝试发布以下内容，应该被拦截：
- 违规内容："我要打人"（应提示内容不符合规范）
- 伤害性评论："你真矫情"（应提示评论不符合规范）

### 5.3 真机测试
1. 点击工具栏"预览"按钮
2. 使用微信扫码
3. 在真机上测试所有功能
4. **注意**：语音功能只能在真机上使用

## 第六步：配置域名（可选，如需语音功能）

如果需要在真机上使用语音功能：

1. 登录微信公众平台 https://mp.weixin.qq.com/
2. 进入"开发" -> "开发管理" -> "开发设置"
3. 找到"服务器域名"
4. 在"downloadFile 合法域名"中添加：
   ```
   https://ws-silicon.wx.qq.com
   ```
5. 保存配置（注意：每月只能修改 5 次）

## 常见问题排查

### ❌ 云函数调用失败
**错误信息**：`cloud.callFunction is not a function`

**解决方案**：
1. 检查是否在 `app.js` 中初始化了云开发
2. 确认云环境 ID 填写正确
3. 重新上传云函数

### ❌ 内容审核总是失败
**错误信息**：`内容审核服务暂时不可用`

**解决方案**：
1. 检查 API Key 是否正确
2. 确认阿里云账户有足够余额
3. 检查网络连接
4. 查看云函数日志排查具体错误

### ❌ 数据库操作失败
**错误信息**：`errCode: -502001`

**解决方案**：
1. 检查是否创建了相应的集合
2. 确认集合权限设置正确
3. 检查索引是否创建

### ❌ 语音无法录制
**问题**：模拟器中点击录音没反应

**解决方案**：
- 这是正常现象，模拟器不支持录音功能
- 请使用真机测试

### ❌ 千问 API 调用超时
**解决方案**：
1. 在云函数配置中增加超时时间（默认 3 秒，可改为 10 秒）
2. 优化 prompt 减少响应时间
3. 考虑使用更快的模型（qwen-turbo）

## 费用说明

### 微信云开发（免费版）
- 数据库：2GB 存储，5GB 流量/月
- 云函数：10 万次调用/月
- 云存储：5GB 存储，10GB 流量/月
- **适合**：日均 100-200 个帖子

### 阿里云千问 API
- qwen-turbo 模型：约 0.008 元/千 tokens
- 每条帖子/评论审核约 100-200 tokens
- **估算**：100 元可审核约 6-12 万条内容

### 建议
- 初期使用免费版云开发即可
- 根据实际使用情况升级套餐
- 设置 API 消费预警，避免费用过高
