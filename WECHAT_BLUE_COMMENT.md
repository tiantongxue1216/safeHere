## ✨ 核心特性

### 1. 微信风格设计

#### 布局方式
- **从左到右**：语音按钮 → 输入框 → 表情按钮 → 发送按钮
- **输入框内**：右侧有语音输入图标（🎤）
- **固定底部**：输入栏固定在页面底部

#### 视觉元素
- **透明圆形按钮**：语音和表情按钮使用透明背景
- **白色圆角输入框**：白色背景，圆角 44rpx
- **渐变发送按钮**：蓝色渐变主题色
- **浅灰背景**：输入栏背景 #f5f5f5

---

## 💡 交互逻辑

### 状态管理

#### 1. 语音按钮状态

```xml
<!-- 正常状态 -->
<text class="voice-icon"></text>
<!-- 显示蓝色语音图标 -->

<!-- 录音状态 -->
<text class="voice-icon recording-icon">🔴</text>
<!-- 显示红色圆点 -->
```

**触发条件：**
- 按压 → 录音 → 显示红点
- 松开 → 停止 → 显示语音图标

---

#### 2. 发送按钮状态

```xml
<!-- 禁用状态 -->
<button class="send-btn" disabled>
  <text>发送</text>
</button>
<!-- 灰色背景，半透明文字 -->

<!-- 激活状态 -->
<button class="send-btn active">
  <text>发送</text>
</button>
<!-- 蓝色渐变，白色文字 -->
```

**触发条件：**
- 有文字或语音 → 激活（蓝色渐变）
- 无内容 → 禁用（灰色）

---

#### 3. 输入框内图标显示

```xml
<!-- 空输入框 -->
<text wx:if="{{!commentContent && !tempCommentVoicePath}}">🎤</text>

<!-- 有内容时隐藏 -->
```

**逻辑：**
- 空输入框 → 显示麦克风图标（提示可以语音输入）
- 有内容 → 隐藏图标

## 🔍 细节优化

### 1. 输入框自动高度

```xml
auto-height
```

**效果：** 随文字内容自动增高，最多 300rpx

---

### 2. 键盘不遮挡

```xml
adjust-position="{{false}}"
```

**作用：** 输入时键盘不自动上推页面

---

### 3. 隐藏确认栏

```xml
show-confirm-bar="{{false}}"
```

**效果：** 输入时不显示键盘确认栏

---

### 4. 录音取消功能

```xml
bindtouchcancel="stopRecordComment"
```

**场景：** 手指滑出按钮区域自动取消录音

---

### 5. 安全区域适配

```css
padding-bottom: calc(16rpx + env(safe-area-inset-bottom));
```

**作用：** 自动适配 iPhone X 及以上机型的底部安全区域
